# 小红书作品封面提取插件 (REDnote Content Scraping)

这是一个基于 [WXT](https://wxt.dev/) 框架和 React 开发的现代浏览器扩展。

它的主要功能是帮助你在**小红书 (REDnote)** 用户的个人主页上自动扫描和提取笔记的封面图片，并支持一键批量下载。

## ✨ 核心特性

- **🔍 自动扫描封面**：自动提取并在页面右下角显示当前小红书主页已扫描到的笔记封面数量。
- **📜 滚动加载支持**：随着用户的不断向下滑动，插件会自动捕捉新加载出来的笔记封面图。
- **📥 一键批量下载**：支持一键批量下载当页扫描到的所有高清封面图，无需手动一张张保存。
- **📁 智能文件归类**：自动获取当前的小红书用户名，下载的图片会自动保存在以用户名字命名的文件夹中。

## 🛠️ 技术栈

- [WXT](https://wxt.dev/) - 次世代浏览器扩展构建工具
- React & TypeScript - UI 渲染及开发语言
- Tailwind CSS / Vanilla CSS - 样式处理

## 🚀 本地开发与使用

### 环境要求

请确保你的本地环境中已经安装了 [Node.js](https://nodejs.org/) (推荐 v18+)。你可以使用 `npm` 或 `pnpm` 作为包管理器。

### 安装依赖

```bash
npm install
# 或者
pnpm install
```

### 启动开发服务器

开发模式下可以自动加载扩展并在浏览器中实时热更新。

```bash
npm run dev
# 或者
pnpm run dev
```

启动后，Chrome 浏览器会自动以开发者模式打开并挂载此插件。

### 构建发布版本

当你开发完成准备打包此扩展时：

```bash
npm run build
# 或者
pnpm run build
```

该命令会在 `.output` 目录生成打包好的扩展文件。你可以将生成的压缩包或者文件夹拖入浏览器的扩展管理页面（如 `chrome://extensions/`）进行使用。

## 💡 使用指南

1. 安装并开启此插件。
2. 浏览器打开任意小红书用户的个人主页 (类似于 `https://www.xiaohongshu.com/user/profile/...`)。
3. 页面的右下角（或浮动容器）会出现“小红书作品采集”面板。
4. 向下滚动页面，让更多的笔记加载出来，插件会自动统计捕捉到的封面数量。
5. 点击 **“批量下载本页封面”** 按钮，浏览器即可自动开始将所有扫描到的图片下载至本地（以用户名命名的子目录中）。
